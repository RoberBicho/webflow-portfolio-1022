Proyectos

Proyectos de software elaborados desde cero con el objetivo de aprender programación web y adentrarme en el mundo del código. Un complemento ideal para no perder el ritmo frenético del mundo en el que vivimos.

Front End App (Git)
Frontend desarrollado en Angular 8 en la cual he creado una web para clínicas. Funcionalidades de reserva de citas con recordatorios para los usuarios, blog, gestión de citas por el administrador y creación de diferentes roles.

RestServer - Node.js - Express (Git)
Backend desarrollado en node.js. Código backend del cual se nutre la app anteriormente presentada. Api desarrollada con métodos get, post, put, delete. También login de google entre otras opciones.

Weather Forecast Simple App (Git)
App simple para la consulta del tiempo por localidades. Consulta a una api proporcionada.

eCommerce - videogames - web (Git)
eCommerce frontend desarrollado en Angular 10 para la compra de videojuegos. Consultas a api propia presentada en el siguiente capítulo.

eCommerce - Videogames - BackEnd - Node, Graphql, Express(Git)

Aipi propia creada desde cero en lenguaje Graphql como complemento y database de eCommerce Videogames web.

eCommerce - ClothingShop Web - Angular(Git)

eCommerce de ropa básica. Mas completo que el proyecto anterior con la dificultad de añadir a carrito talla y color. Las consultas a la api adquieren mayor complejidad. Consultas hechas al siguiente proyecto de la lista.

eCommerce ClothingShop BackEnd - Node, Graphql, Express(Git)

Api creada desde cero para nutrir a su respectivo eCommerce web. Dificultad de crear filtros para la consulta y guardar objetos de características complejas. Herramientas usadas como stripe, mongoDb, google api entre otras.
